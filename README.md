# 4 Months Anniversary Website

A small romantic static website for a 4-month anniversary. It is ready to publish with GitHub Pages.

## Customize

Open `script.js` and update these values:

- `partnerName`: your girlfriend's name
- `startDate`: the date you started dating
- `startDateLabel`: the date shown on the page
- `memories`: the memory cards and image links

You can also edit the love letter directly in `index.html`.

## Preview Locally

Open `index.html` in a browser.

## Publish On GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, `script.js`, and `README.md`.
3. Go to repository `Settings` > `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Choose the `main` branch and `/root`, then save.

Your site will be available at:

```text
https://your-github-username.github.io/your-repository-name/
```
